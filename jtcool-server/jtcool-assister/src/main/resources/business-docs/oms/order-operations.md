# 订单操作指南

## 创建订单

### 基本信息
创建订单需要提供以下信息：
- 客户ID (customerId)
- 商品列表 (items)
  - 商品ID (productId)
  - 数量 (quantity)
  - 单价 (price)
- 收货地址 (deliveryAddress)
- 联系电话 (contactPhone)

### 示例
```json
{
  "customerId": 100,
  "items": [
    {"productId": 200, "quantity": 10, "price": 99.00}
  ],
  "deliveryAddress": "北京市朝阳区xxx",
  "contactPhone": "13800138000"
}
```

## 查询订单

### 按订单ID查询
提供订单ID即可查询订单详情，包括：
- 订单基本信息
- 当前状态
- 商品明细
- 操作历史

### 按条件查询
支持的查询条件：
- 客户ID
- 订单状态
- 创建时间范围
- 订单金额范围

## 更新订单状态

### 推进状态
使用"推进订单状态"功能，系统会：
1. 验证当前用户权限
2. 检查状态转换是否合法
3. 执行状态转换
4. 记录操作历史

### 注意事项
- 只能推进到下一个允许的状态
- 需要相应的角色权限
- 状态转换会记录操作人和时间

## 订单取消

只有在以下状态可以取消：
- 已下单 (PLACED)
- 销售确认 (SALES_CONFIRMED)

取消后订单状态变为"已取消"，不可恢复。
