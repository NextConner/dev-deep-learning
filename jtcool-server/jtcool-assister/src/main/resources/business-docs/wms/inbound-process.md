# 入库流程

## 采购入库

### 流程步骤
1. **创建入库单**：采购订单到货后，仓库创建入库单
2. **验货**：检查商品数量、质量是否符合采购订单
3. **上架**：将商品放入指定货位
4. **确认入库**：系统增加库存数量

### 入库单信息
- 入库单号（系统自动生成）
- 供应商信息
- 商品明细（商品ID、数量、单价）
- 入库仓库
- 入库时间
- 操作人员

## 退货入库

### 流程步骤
1. **客户申请退货**：客户服务审核通过
2. **创建退货入库单**：关联原订单
3. **收货验货**：检查退货商品状态
4. **质检**：判断是否可以重新销售
5. **上架或报损**：合格品上架，不合格品报损

### 质检标准
- 包装完好
- 商品无损坏
- 在保质期内

## 调拨入库

### 流程步骤
1. **接收调拨通知**：其他仓库发起调拨
2. **收货**：确认收到调拨商品
3. **验货**：核对数量和商品信息
4. **上架**：放入货位
5. **确认入库**：更新库存

## 注意事项
- 入库前必须验货，发现问题及时反馈
- 入库单必须与实际到货一致
- 入库后及时更新库存系统
