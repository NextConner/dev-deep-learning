# 库存管理指南

## 库存查询

### 按商品查询
提供商品ID查询该商品在所有仓库的库存情况：
- 可用库存
- 锁定库存（已分配给订单但未出库）
- 总库存

### 按仓库查询
提供仓库ID查询该仓库所有商品的库存情况。

## 库存变动

### 入库
- 采购入库：供应商送货入库
- 退货入库：客户退货入库
- 调拨入库：从其他仓库调入

### 出库
- 销售出库：订单发货出库
- 报损出库：商品损坏报废
- 调拨出库：调往其他仓库

### 库存锁定
- 订单确认后自动锁定库存
- 发货后扣减实际库存
- 订单取消后释放锁定库存

## 库存预警

### 低库存预警
- 库存数量 < 安全库存：发送预警通知
- 建议补货数量 = 最大库存 - 当前库存

### 超库存预警
- 库存数量 > 最大库存：提示库存积压
- 建议促销或调拨

## 库存盘点

### 盘点流程
1. 创建盘点单
2. 实地盘点并录入实际数量
3. 系统对比账面数量和实际数量
4. 生成盘盈盘亏报告
5. 审核后调整库存

### 盘点周期
- 日常盘点：每月一次
- 年度盘点：每年一次（全面盘点）
