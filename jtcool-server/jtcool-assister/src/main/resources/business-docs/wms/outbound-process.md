# 出库流程

## 销售出库

### 流程步骤
1. **接收出库指令**：订单状态到达"仓库确认"时自动创建出库单
2. **拣货**：根据出库单从货位拣取商品
3. **复核**：核对商品和数量是否正确
4. **打包**：按照配送要求打包
5. **登记出库**：扫描出库，系统扣减库存
6. **交接物流**：将包裹交给物流公司

### 出库单信息
- 出库单号（系统自动生成）
- 关联订单号
- 商品明细
- 收货地址
- 物流方式
- 出库时间

## 拣货策略

### 先进先出（FIFO）
- 优先拣取入库时间早的商品
- 适用于有保质期的商品

### 就近原则
- 优先拣取距离近的货位
- 提高拣货效率

## 出库异常处理

### 库存不足
- 立即通知采购部门补货
- 联系客户协商延期发货或部分发货

### 商品损坏
- 拣货时发现商品损坏，更换其他货位的商品
- 损坏商品做报损处理

### 地址错误
- 联系客户确认正确地址
- 修改出库单信息后再发货
