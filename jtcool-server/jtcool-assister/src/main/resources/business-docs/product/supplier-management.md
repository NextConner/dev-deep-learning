# 供应商管理

## 供应商信息

### 基本信息
- 供应商名称
- 供应商编码
- 联系人
- 联系电话
- 地址
- 邮箱

### 资质信息
- 营业执照
- 税务登记证
- 银行账户信息

## 供应商评级

### 评级标准
- A级：优质供应商，交货准时，质量稳定
- B级：合格供应商，偶有延迟或质量问题
- C级：待改进供应商，问题较多
- D级：不合格供应商，建议更换

### 评级依据
- 交货准时率
- 产品合格率
- 价格竞争力
- 服务响应速度

## 供应商合作

### 采购流程
1. 创建采购订单
2. 发送给供应商
3. 供应商确认
4. 供应商发货
5. 仓库验收入库
6. 财务付款

### 结算方式
- 现金结算
- 月结（账期30天、60天、90天）
- 预付款

## 供应商考核

### 考核周期
- 月度考核
- 季度考核
- 年度考核

### 考核指标
- 交货及时率 ≥ 95%
- 产品合格率 ≥ 98%
- 服务满意度 ≥ 90%
