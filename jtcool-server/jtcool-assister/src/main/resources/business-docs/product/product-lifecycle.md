# 商品生命周期管理

## 商品创建

### 基本信息
- 商品名称
- 商品编码（唯一）
- 品牌
- 分类
- 规格型号
- 单位（件、箱、kg等）

### 价格信息
- 成本价
- 销售价
- 会员价
- 批发价

### 库存信息
- 安全库存
- 最大库存
- 初始库存

## 商品状态

### 在售（ON_SALE）
- 商品正常销售
- 可以下单购买

### 下架（OFF_SHELF）
- 暂停销售
- 不可下单
- 原因：库存不足、质量问题、季节性下架等

### 停产（DISCONTINUED）
- 永久停止销售
- 清理剩余库存

## 商品维护

### 价格调整
- 需要审批权限
- 记录价格变更历史
- 生效时间可设置

### 库存调整
- 盘点后调整
- 报损报溢
- 记录调整原因

### 商品信息更新
- 修改商品描述、图片
- 更新规格参数
- 保留变更记录
